**Loss of Control** is a standalone backport of the modern WoW Loss of Control indicator for `Legacy` and `Classic Era` versions.

It displays crowd control (CC) effects (Stun, Fear, Silence, Disarm, etc.) with an icon, remaining duration, and a cooldown spiral.

## ✨ Features

*   **Retail Experience:** Faithful recreation of visuals and animations.
*   **Accurate Timers:** Handles interrupt lockouts (Kick, Counterspell) correctly.
*   **Customization:** Addon has its own visual settings system.
*   **Localized:** `enUS`, `ruRU`, `deDE`, `frFR`, `esES`, `zhCN`, `zhTW`, `koKR`.

## 📂 Installation

1.  Download the **[Latest Release](https://github.com/s0h2x/LossOfControl/releases/latest)** (zip file).
2.  Extract `LossOfControl`.zip into `World of Warcraft\Interface\AddOns\` directory.
3.  After extraction, the addon must be located here: `Interface\AddOns\LossOfControl\LossOfControl.toc`.
4.  Restart World of Warcraft.

## ⚙️ Commands

Settings available via `/loc` or `/los`.

| Command | Action |
| :--- | :--- |
| `/loc` | Open main settings |
| `/loc auras` | Open Aura Manager (your own custom auras) |
